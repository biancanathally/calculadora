# Calculadora Python

Projeto para disciplina de **Fundamentos de Testes Ágeis** da Especialização Lato Sensu em Testes Ágeis, no CESAR School.

Calculadora com interface gráfica (GUI) desenvolvida em Python utilizando a biblioteca **Tkinter**. A calculadora segue um design clássico de botões em grelha, mas está configurada para trabalhar especificamente com os números **1, 2, 3 e 4**.

## 📋 Pré-requisitos

Para rodar este projeto, precisa apenas ter o **Python 3** instalado. A biblioteca `tkinter` já vem instalada na maioria das distribuições de Python.

## 🔧 Como rodar o projeto

1. **Clonar ou copiar o código:**
   Criar um ficheiro no seu computador chamado `calculadora.py` e colar o código da calculadora dentro dele.

2. **Abrir o terminal:**
   Navegue até à pasta onde guardaste o ficheiro.

3. **Executar o programa:**
   Digite o seguinte comando no terminal (ou linha de comandos):

   ```bash
   python calculadora.py
